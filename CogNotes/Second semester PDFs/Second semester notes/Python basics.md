# Working directories and movement from console

| Command              | Meaning                                                          |
| -------------------- | ---------------------------------------------------------------- |
| **ls**               | list directory content (what can you see in your current folder) |
| **pwd**              | print working directory (what folder are you currently in)       |
| **cd**               | change directory                                                 |
| **touch**            | create files                                                     |
| **rm**               | remove files                                                     |
| **type**             | get the type of varriable                                        |
| **python3 -m venv** name | Create virtual enviorment                                        |
